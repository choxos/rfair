# Summarize a FAIR assessment as an F/A/I/R score table.

Summarize a FAIR assessment as an F/A/I/R score table.

## Usage

``` r
# S3 method for class 'fair_assessment'
summary(object, ...)
```

## Arguments

- object:

  A `fair_assessment` object.

- ...:

  Ignored.

## Value

A data frame with earned, total, percent, and maturity per FAIR category
and overall.
