# Convert a FAIR assessment to a per-metric data frame.

Convert a FAIR assessment to a per-metric data frame.

## Usage

``` r
# S3 method for class 'fair_assessment'
as.data.frame(x, ...)
```

## Arguments

- x:

  A `fair_assessment` object.

- ...:

  Ignored.

## Value

A data frame with one row per metric.
